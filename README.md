## 📋 About
This project documents **Exposed Apple Products zero-day vulnerabilities** from 2020 to 2025, including:
- **CVE IDs**
- **Affected components**
- **Impact and exploit status**
- **Mitigation advice**
- **References to official advisories and security news**
